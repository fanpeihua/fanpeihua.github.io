Harness Books — Claude Code & Codex 设计哲学

两本关于 coding agent harness 工程的书，作者 wquguru。原文阅读站托管在 AgentWay。

官方在线阅读 GitHub 源 English Version

当一个会写代码的模型被放进终端、仓库、权限系统、团队工作流——什么让整个系统保持有界、连贯、可问责？这两本书不逐行讲源码，而是讨论一个 harness 如何组织约束与执行，如何把"模型必然不稳定"折叠成可持续的工程秩序。
